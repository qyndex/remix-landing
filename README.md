# Remix Landing

Marketing landing page with hero, features grid, and pricing tiers using Remix 2 loaders and React 19.

**Stack:** Remix | **Complexity:** simple
**Tags:** `remix`, `react`, `landing`, `marketing`, `hero`, `pricing`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
